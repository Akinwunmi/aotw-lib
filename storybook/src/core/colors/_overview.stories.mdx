# AOTW Colors

Archive of the World color tokens for _CSS_, _Sass_ and _SCSS_.

### Color token structure

`aotw-color-[type]-[range]`

#### Examples

- `var(--aotw-color-tertiary-300)` (CSS)
- `$aotw-color-error-500` (Sass / SCSS)

### Palette structure

| Name      | Default color | Range       |
|-----------|---------------|-------------|
| primary   | blue          | `50 - 900` |
| secondary | grey          | `50 - 900` |
| tertiary  | wheat         | `300 - 500` |
| success   | green         | `500`       |
| error     | red           | `500`       |
| warning   | orange        | `500`       |
